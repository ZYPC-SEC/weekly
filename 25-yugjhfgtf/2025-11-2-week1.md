# <font style="color:rgb(38, 38, 38);">本周学习的知识分享</font>

在C语言中，赋值是一种静态的行为，赋值变量和被赋值变量的关系在赋值完成后就结束了

但是在Python中则有很大不同，分为直接赋值，浅拷贝，深拷贝

| 种类     | 区别                                     |
| :------- | ---------------------------------------- |
| 直接赋值 | 就是同一个，赋的两个变量指向同一个对象   |
| 浅拷贝   | 本身不相同，但内嵌的对象可能指向相同对象 |
| 深拷贝   | 完全不相同，就算长得一样也是纯粹的复制   |

在这一点上，C语言的赋值类似于复制，Python的赋值则更相似于某种对“对象”的引用
