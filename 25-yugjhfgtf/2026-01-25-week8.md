## *椭圆曲线ECC*

##### 在说椭曲前我们需要一些基础知识

```
群论
研究抽象的学科，群环域这里我们只用知道群
```

```
群(Group)
非常字面，一堆东西构成一个群
具体来讲是一堆元素和一个运算，可以非常不代数
（一个正方形顺时针旋转而形状不变的角度）构成一个群
元素就是90',180',270',360',运算就是顺时针旋转
```

```
群的判定
判断一堆东西构成群与否，我们有四条规则
1.满足封闭律
即群内元素与群内元素用构成群那个运算只能得到群内元素
2.满足结合律
字面意，就是小学学的那个
3.拥有单位元
一个元素e满足任意群内元素a @ e = a，e就是单位元，这里@拿来代表下运算
4，元素有逆元
如a @ a^-1 = e ，也就是元素有逆元，元素和逆元运算得到单位元，眼熟吗？
```

嗨呀好麻烦

```
阿贝尔群(Abelian Group)
群，但是还满足第五条规则
5.满足交换律
就这样，很简单吧
```

```
朴素版的椭圆曲线公式，还有判别式
y^2 = x^3 + ax + b

```

这个判别式用来判断曲线是否***奇异***
简单来说就是曲线上有无**奇点**(singularity),一个不奇异的曲线上任意一个点有且仅有一条切线

![space-1.jpg](https://pic1.imgdb.cn/item/697607116deeadc41a33aa69.png)

首先取一个点A和一个点B，两点连线画直线，与曲线交于点C‘，作与x轴对称的点C
好了，这个A@B=C的过程被我们定义出来了
![](https://pic1.imgdb.cn/item/69760a746deeadc41a33aad6.png)

神奇的是，这个运算(在曲线非奇异时)符合我们的五条规则，因此构成了阿贝尔群
